##  Setup Ambiente

```bash
# clonare il corso
git clone https://github.com/danielino/python-course-jupyter
cd python-course-jupyter

# creazione ed attivaizone virtualen
python3.7 -m venv venv

# installazione dipendenze
[[ -f requiremments.txt ]] && \

jupyter lab
```
